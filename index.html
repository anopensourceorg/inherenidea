Benefits of joining the programme

  • Hands-on real world projects
  • Certificate of completion
  • Flexible Timings
  • Incentives on good performance

More about programme

  • Developers would be given the opportunity to work on real-world projects of web development under proper guidance and mentorsip
  • Latest technologies like react js and node js would be used in the projects
  • Weekly competitive contest would be organized where developers can compete and enhance their skillsets
  • Mentorships sessions would be organized for developers to guide them about the technologies they would be working on

Role of a developer

  • Each developer will form an important node for our network
  • Developer will be supposed to work on a feature divided into sub tasks.
  • Developer can choose the feature on which he/she is interested to work on.
